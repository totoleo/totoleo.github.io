这是我目前找到的最好的遍历字符串的方法。可是我在想，能否使用`yield`来遍历数组，这样对长字符串来说，可以减少内存使用了。
下面就是我的实验代码了
```php
$str = 'hello world';
$chars = string_rangex($str);
foreach($chars as $char){
  echo $char.PHP_EOL;
}

function string_rangex($str){
  $len = strlen($str);

  for($i = 0;$i < $len;$i++){
    yield substr($str,$i,1);
  }
}
```
